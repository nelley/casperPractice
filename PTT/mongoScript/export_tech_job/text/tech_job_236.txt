大家好 距離這篇自學文也快要兩年時間了。
想報告下近況!

2016年中每天下班用在Free Code Camp自學。。學了幾個月後把前端學完後拿到了個前端
證照就算告一段落。。。FCC後面的課程2016年那時感覺系統性沒有前端課程來得有系統
所以沒有繼續FCC。

但,在寫程式方面雖然還是個新手但對於自學這件事感到有興趣。

講到自學順帶一提2017年初開始Duolingo學法文。後來把Duolingo的法文課程全部學完了
。後來和男友去歐洲旅遊時在法國可以認路和餐廳點餐。目前還是每天會花15分鐘簡單復
習法文。。

回歸正題。這段時間我都還是在同公司。 公司似乎發現我有ㄧ點點寫程式底子。2017年
公司一年內給我加了兩次薪。ㄧ次8%另ㄧ次4% 開始讓我和軟體開發部門software develo
pment 合作。有新軟體或新功能時會找我測試。後來也讓我訓練沒接觸程式但的員工簡單
接觸使用程式(其實只是教他們至少要會放入parameters然後運行之類的)。然後我也幫忙
軟體部門負責開發Machine Learning model的工程師準備數據(其實超累。。但是因為都
是聽工程師的指揮去做,讓我稍為了解對於準備machine learning model需要的流程)


2017年十月左右。我開始上網查還有什麼能學的。我發現我使用的學習程式的共同點都是
介面可愛好上手。字體清楚,解釋的淺顯易懂。所以找的也是這方面的資源。一開始我都
想找免費軟體。。。後來找到了DataCamp 學的是Data Scientist with Python 。。剛好
跟我2017年時幫工程師準備數據這方面有點相關 但不是免費的所以一開始有點抗拒 試用
算一算價錢勉強可接受 就訂閱了一年份(加幣兩百多。。) 然後之後每天下班兩小時自學
。 1月份時也學完了Data Scientist with Python的20門課程。現在就是練習 然後把Mac
hine learning弄得更懂些之後還打算接觸SQL



然後。。。今年初公司終於終於正式把我的轉成研發部門了!!軟體開發部門的主管很欣賞
我,他手下的員工有幾個是我臉書好友。所以可能有讓他知道我有自學習慣,他要我開始做
machine learning developement 和前輩ㄧ起做一起學習。所以會有人帶我並且部門會撥
出資源加強訓練我的寫程式能力。也就是說我也可以用在公司的時間自學。或是網上自學
花的錢 買書的錢或是參加研討會也都可報帳。

我好開心!! 雖然兩年前是接觸前端,現在做的是開發機器學習的工作,但其實比起前端這
更是我想要做的!!而且感覺自己的努力是有被認可的!!!而且學到的技能是自己的不怕失
業,就算失業因為只要有技能也容易找下份工作。

學習真的不嫌晚 什麼都是可以學的!!學的慢什麼的都沒關係! 我現在相信只要真正想要
一件事 努力去做 就真的很可能會擁有的!!

ps如果有人是機器學習這方面的前輩 如果可以的話能分享給我更多能自學的資源就太好
了!!
